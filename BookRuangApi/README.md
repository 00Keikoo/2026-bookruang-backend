# BookRuang Backend

## Description
Backend API untuk sistem manajemen RoomLoan dengan ASP.NET Core dan SQLite

## Features
- Buat peminjaman ruang
- Lihat daftar pinjaman
- Lihat detail pinjaman
- Update pinjman
- Hapus pinjaman

# Tech Stack 
- ASP.NET Core 8 
- Entity Framework Core
- SQLite

## Installation
``` bash
dotnet restore
dotnet build 
dotnet run

# Usage
Open Localhost/Swagger

# Environment 
No .env required. Using SQLite local database
